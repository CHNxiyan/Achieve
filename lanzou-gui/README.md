# 说明
- 默认并发上传下载任务为3，可以自行设置，单个文件还是单线程的；

- 文件可以直接拖拽到软件界面上传，也可以使用对话框选择；

- 文件夹最多**4级**，这是蓝奏云的限制；

- 文件上传后不能改名，同时最好不要创建相同名字的文件夹；

- 更多说明与详细界面预览详见 [WiKi](https://github.com/rachpt/lanzou-gui/wiki)。
